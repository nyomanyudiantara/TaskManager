🚀 Tech Stack
-------------

-   **Frontend:** React, TypeScript, Vite, React Router

-   **Backend:** Node.js, Express.js

-   **Database:** MySQL

-   **Authentication:** UUID-based Session Management

-   **Styling:** CSS-in-JS (Inline Styles) with Archivo Font
